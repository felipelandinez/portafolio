Mi perfil

Este proyecto es una página simple creada con HTML y CSS, diseñada para mostrar mi perfil personal. Utiliza HTML para la estructura y CSS para el diseño, asegurando una experiencia visual atractiva y responsiva en distintos dispositivos.

Quién Soy

Este proyecto es una página web que muestra quién soy, mis habilidades y algunos de los proyectos que he realizado. Está diseñada con HTML y CSS para ofrecer una presentación clara y atractiva de mi perfil profesional.

Página de Conciertos

Este proyecto es una página web que simula una cartelera de conciertos, donde se muestran eventos y detalles. Está diseñada con HTML y CSS para crear una experiencia interactiva y dinámica.

Video: Cómo Desplegar

Hoja de vida

Este es un proyecto de mi hoja de vida. A través de esta aplicación, los usuarios pueden explorar de manera dinámica mi experiencia profesional, los idiomas que hablo y mis habilidades.

Tienda Electrónica

Este proyecto muestra una tienda electrónica con productos destacados, diseñada usando HTML y CSS para una experiencia moderna y responsiva.

App de citas veterinarias

Este es un proyecto de mi portafolio. Con esta aplicación, los usuarios pueden gestionar citas veterinarias de forma sencilla, filtrarlas y mantener todo organizado desde una interfaz clara y funcional.

Pokedex

Este proyecto es una Pokédex interactiva que muestra un Pokémon aleatorio usando HTML, CSS y JavaScript, con datos obtenidos de la PokéAPI. Su diseño simula el estilo de una Pokédex clásica.

Talonario

Este proyecto es un talonario de rifa interactivo desarrollado con Vue.js y JavaScript. Permite gestionar boletos de forma dinámica, mostrando información de cada número en una interfaz moderna y funcional.

Hotel Monte verde

Este es un proyecto de una página web para un hotel. Los usuarios pueden consultar información sobre el alojamiento, explorar las habitaciones disponibles, realizar reservas en línea, todo desde una interfaz intuitiva, moderna y fácil de usar.
